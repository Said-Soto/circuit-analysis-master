# Circuit Analysis Master

App interactiva para practicar análisis de circuitos paso a paso.

## Temas Incluidos
- Análisis de Mallas (3 ejercicios)
- Análisis de Nodos (3 ejercicios)
- Teorema de Thévenin (3 ejercicios)
- Transitorios RL (3 ejercicios)
- Transitorios RC (3 ejercicios)

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Stitch AI (Design System)

## Cómo usar
Simplemente abre `index.html` en tu navegador.
